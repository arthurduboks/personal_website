behind the script.

In summary, the script displays a cookie box pop-up to inform the user about cookie usage. If the user clicks Accept, a cookie is set, the cookie box is hidden, and the Microsoft Clarity tracking code (if cookies are accepted) is loaded. If the user clicks Decline, any existing cookies are removed, the cookie box is hidden, and the tracking code is optionally disabled. The script ensures that the cookie box is shown only if the user has not accepted cookies before🙈
